StrategyBlocksLight

Welcome to the Homepage for the StrategyBlocks lightweight front-end.

Warning: This library is still a work in progress

This library is intended to be available for free for anyone who wants to develop their own client for StrategyBlocks. This library is being used as the backbone for the IPhone app, and is being actively developed for use in a new HTML5 StrategyBlocks client.

To use this library effectively, you will need to create a StrategyBlocks account if you don't already have one.

Once you have an account, check out some of the html files in the /tests folder and see some basic examples for how to use it. There's no documentation yet, but feel free to post any problems/questions under the issues section of the repository and I'll get back to you asap.
